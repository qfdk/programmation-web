### git 基本操作命令

```bash
# 把字符串写入到 README 文件里面
echo "# programmation-web" >> README.md
# 初始化 文件夹 （只有第一次使用）
git init
# 只添加 README 文件 不会添加其他的文件
git add README.md
git commit -m "第一次提交的信息或者注释"
git branch -M main
git remote add origin https://github.com/qfdk/programmation-web.git
git push -u origin main
```
